Research

Multimodal Emotion Detection

This research focuses on building a multimodal sentiment analysis system using audio and video modalities to monitor emotional states for remote mental health therapy. Utilizing the CREMA-D dataset, a deep learning pipeline was developed that achieved up to 84% accuracy in emotion classification.

Methods used included CNNs for facial feature extraction, MFCCs for audio signal processing, and fusion techniques to combine modalities. This work was part of my undergraduate thesis and has practical applications in teletherapy, digital well-being, and AI-driven health monitoring.

Multi-Crop Leaf Disease Detection

Designed and developed a deep learning pipeline to detect leaf diseases across multiple crops—Rice, Corn, Wheat, Potato, and Sugarcane—using image classification and computer vision.

Built a full-stack crop disease detection application using Python, FastAPI, and Next.js. The core model was based on CNN architectures fine-tuned for each crop type, providing reliable disease classification through mobile/web-based image capture.
